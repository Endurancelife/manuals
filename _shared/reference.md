# 4. Flexible Alerts Plugin

These alerts are created using blockquotes and alert keywords.

> [!NOTE]
> **This is a Note**  
> Use this blue box for general information and reminders.

> [!TIP]
> **This is a Tip**  
> Use this green box for helpful advice and best practices.

> [!WARNING]
> **This is a Warning**  
> Use this yellow box to highlight important cautions.

> [!DANGER]
> **This is a Danger Alert**  
> Use this red box for critical safety or emergency information.
